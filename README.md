## Como executar
1. `git clone https://github.com/luis-antoniosi/web-potion-seller.git`
2. `cd web-potion-seller`
3. `npm install`
4. `npm run start`
5. Servidor abre na porta 8080; entrar em http://localhost:8080

## Endpoints
`/potions`       -> **GET**, retorna todas as poções.<br/>
`/potions/:id`  -> **GET**, retorna uma poção por id.<br/>
`/potions`      -> **POST**, cria uma poção utilizando os atributos do corpo da requisição.<br/>
`/potions/:id`  -> **DELETE**, remove a poção que tem o id fornecido.<br/>
`/potions/:id`  -> **PUT**, atualiza a poção que tem o id fornecido, utilizando os atributos do corpo da requisição (não implementado no site, mas o endpoint funciona).
